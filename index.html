REPOTECH

On the user interface, any registered user can send requests for projects, feasibilities, manage their pending requests and projects, contact administrator for anything, and much more.

On the administrator side, all admins can access to informations, depending on their role, including project management, pending requests, users list...

Every information can be extracted from website to a tab file format like XLSX or CSV.

This application is used by all laboratories within the Renatech Network.

The app is also able to generate stats and graphics about various data related to its database.


In 2023, the REPOTECH app counts :

  • Users : 3 500+
  • Projects submitted in 2022 : 300
  • Pending projects in 2022 : 920
  • Projects submitted since the first deployment : 2 300+
  • Projects with foreign partners : 680
CROSS

On the clean room side, any registered user can access to tools they have right to use. The system will record the products and duration of the experiment.

On the administrator side, all admins can access to information, depending on their role, including logs, experiments duration, prices, users...

Every information can be extracted from website to a tab file format like XLSX or CSV.

In 2023, the CROSS app will manage :

  • Boxes 14
  • Tools 50
  • Operation by year 30 000


Currently, this project is in production in the IEMN lab in Villeneuve d'Ascq.
